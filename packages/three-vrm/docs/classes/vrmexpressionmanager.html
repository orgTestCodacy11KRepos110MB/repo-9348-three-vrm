getExpressionTrackName

  • getExpressionTrackName(name: string): null | string
    • Defined in packages/three-vrm-core/types/expressions/VRMExpressionManager.d.ts:115

    Get a track name of specified expression. This track name is needed to manipulate its expression via keyframe animations.

    example

    Manipulate an expression using keyframe animation

    const trackName = vrm.expressionManager.getExpressionTrackName( 'blink' );
const track = new THREE.NumberKeyframeTrack(
  name,
  [ 0.0, 0.5, 1.0 ], // times
  [ 0.0, 1.0, 0.0 ] // values
);

const clip = new THREE.AnimationClip(
  'blink', // name
  1.0, // duration
  [ track ] // tracks
);

const mixer = new THREE.AnimationMixer( vrm.scene );
const action = mixer.clipAction( clip );
action.play();

    Parameters

    • name: string

      Name of the expression

    Returns null | string
