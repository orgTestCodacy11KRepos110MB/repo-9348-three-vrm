Update a constraint. If there are other constraints that are dependant, it will try to update them recursively. It might throw an error if there are circular dependencies.

Intended to be used in update and _processConstraint itself recursively.

param

A constraint you want to update

param

Set of constraints that are already tried to be updated

param

Set of constraints that are already up to date
